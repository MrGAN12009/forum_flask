# Changelog

Все значимые изменения в проекте документируются здесь.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/).

## [1.0.0] - 2024-01-01

### Добавлено
- Система регистрации и авторизации пользователей
- Flask-Login для управления сессиями
- Хеширование паролей с помощью Werkzeug
- Проверка уникальности email и username
- Форум с топиками и постами
- Создание, просмотр и удаление топиков
- Пагинация для списка топиков и постов
- Счетчик просмотров топиков
- Загрузка изображений в постах
- WebSocket чат между пользователями в реальном времени
- Поддержка отправки изображений в чате
- Индикатор "печатает..."
- Счетчик непрочитанных сообщений
- Профили пользователей с аватарами
- Редактирование профиля
- Изменение пароля
- Статистика пользователя (топики, посты)
- PostgreSQL база данных
- Alembic миграции для управления схемой БД
- Docker и Docker Compose для развертывания
- Nginx reverse proxy
- Конфигурация для HTTPS с Let's Encrypt
- Автоматическое обновление SSL сертификатов
- Gunicorn WSGI сервер с eventlet
- Адаптивный дизайн с Bootstrap 5
- Защита от CSRF
- Валидация загружаемых файлов
- Ограничение размера загрузок (16MB)
- Security headers в Nginx
- Полная документация (README, QUICKSTART)
- Makefile для удобного управления
- Скрипт инициализации БД с тестовыми данными
- Скрипт получения SSL сертификатов

### Технологии
- Flask 3.0.0
- Flask-SQLAlchemy 3.1.1
- Flask-Login 0.6.3
- Flask-SocketIO 5.3.5
- Flask-Migrate 4.0.5
- PostgreSQL 15
- Nginx (Alpine)
- Docker & Docker Compose
- Bootstrap 5.3.0
- Socket.IO 4.5.4
